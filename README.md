# jesting

Api testing for the jpcp using the jest framework

# Installation

1) Clone directory
2) Run npm install

# Configuration

1) Open "testConfig.js"
2) Change "serverUrl" and "port" to match your system's configuration

# Running

1) npm test/n
or/n
2)npm debug //Outputs additional debugging information
